## UART interface

Baudrate: 9600

|   Command   | action                       |
| :---------: | :--------------------------- |
| 0xa0 ~ 0xa3 | set gain                     |
| 0x10 ~ 0x1f | set channel 1 to sensor 1~16 |
| 0x20 ~ 0x2f | set channel 2 to sensor 1~16 |
| 0x30 ~ 0x3f | set channel 3 to sensor 1~16 |
| 0x40 ~ 0x4f | set channel 4 to sensor 1~16 |
